# Виставка «Трансформації» — Project Documentation

## Overview
Односторінковий сайт мистецької виставки «Трансформації».  
Двомовний (UA / EN). Аудиторія — відвідувачі виставки, партнери, ЗМІ.  
Головна цінність: мінімалістичний дизайн, швидке завантаження, коректне відображення на мобільних.

**Посилання:** https://exhibition-transformations.vercel.app  
**EN-версія:** https://exhibition-transformations.vercel.app/en  
**GitHub:** https://github.com/chumdesign/exhibition-transformations

## Forbidden Patterns
- ❌ **Не видаляти** файли зображень без підтвердження — імена містять метадані
- ❌ **Не вмикати SSR** без потреби — сайт статичний, деплоїться як HTML
- ❌ **Не змінювати формат імен файлів** `Автор_Назва_Деталі.webp` — порушить парсинг у `Artworks.astro`
- ❌ **Не використовувати Netlify** — вичерпані ліміти
